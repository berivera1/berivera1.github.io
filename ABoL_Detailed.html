This is a remix of the popular Scratch game A Bit of Luck. The original game is a platformer in which the player must maneuver around spike traps to advance to the next level. The player was a blob character, there was a single song in the soundtrack, the textures were basic, the background was only a simple gradient, the game only had 7 levels, and there was only one world. In my remix, I changed the player to a fish character with more diverse animations, there are over 10 songs in the soundtrack, the textures have been updated to add a more fitting aesthetic, the background has been updated to fit the world, the game has been increased to 15 levels, the game includes 3 worlds, there is a new added block type known as the Teleporter (which is pretty self-explaintory), there is a new and aesthetically pleasing menu screen, there is a usable melee weapon (there was no combat in the original) known as Vox's Reaper, and there is a final boss battle (a mech which shoot rockets at you) named The Silencer.
